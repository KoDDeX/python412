Emmet и его возможности

Простые команды

  1. ! - базовая структура документа
  2. h1{Текст} - создание заголовка первого уровня
  3. h2{Текст}+p{Текст} - создание заголовка второго уровня и абзаца
  4. ul>li*3{$. Элемент списка} - создание списка из трех элементов. Будет присутствовать нумерация

Сложные команды

p.red-text{Привет!} - создание абзаца с классом red-text и текстом "Привет!"

span.red-text>code{print("Hello, World!")} - создание спана с классом red-text и вложенным тегом code и текстом "print("Hello, World!")"

h2{Вставка изображения в HTML}+img[src="1.jpg" alt="Фотография пользователя"] - создание заголовка второго уровня и вставка изображения с атрибутами src и alt

3 способа стилизовать HTML

1. Встроенные стили - <h2 style="background-color: lightgreen;">3 способа стилизовать HTML</h2>

2. Внутренние стили - <style> h2 { background-color: lightgreen; } </style>

3. Внешние стили - <link rel="stylesheet" href="style.css">
